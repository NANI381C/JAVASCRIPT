# JAVASCRIPT ⚡

Welcome to the **JAVASCRIPT** repository! This project contains a collection of interactive HTML and JavaScript files demonstrating core programming concepts, DOM manipulation, and browser APIs.

## 📖 About This Repository
This repository serves as a hands-on workspace for learning vanilla JavaScript. It covers essential topics ranging from basic syntax and string literals to handling user interactions and persisting data using the browser's Local Storage.

## 🗂️ Concepts & Files Covered

### Event Handling
* **Event Listeners:** Modern event attachment (`Add Event listener.html`).
* **Inline Events:** Traditional HTML attribute-based event handling (`inlineeventhandler.html`).

### Data Persistence (Web Storage)
* **Local Storage API:** Storing and retrieving data in the browser (`localstorage.html`).
* **Login Flow Simulation:** A practical application of local storage to simulate user authentication and redirection (`login.html`, `welcome.html`).

### Core Concepts & Logic Examples
* **String Literals:** Working with modern template literals (`literals.html`).
* **Practical Examples:** Logic building and structured examples, such as a simulated bank account interaction and inner/outer function demonstrations (`Example1.html`, `Example1(inner&Outer).html`, `Example2(Bank Account).html`).
* **Foundations:** Basic JavaScript implementation (`javascript.html`).
